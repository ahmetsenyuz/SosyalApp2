# SosyalApp2

A social media application for connecting users and sharing content.

## Overview

This project is a social media platform that allows users to create accounts, share posts, follow other users, and interact with content through likes and comments.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

- .NET SDK 6.0 or later
- Visual Studio or VS Code

### Installation

1. Clone the repository
2. Restore dependencies: `dotnet restore`
3. Build the project: `dotnet build`
4. Run the application: `dotnet run`
