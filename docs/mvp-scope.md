# SosyalApp2 MVP Scope Document

## Objective
Establish the minimum viable product scope by identifying core functionalities that must be included for the primary user journey to be supported, ensuring alignment with the project's social task and scoring tracking purpose.

## Core Features (Must-Have)
1. User Registration and Authentication
2. Profile Management
3. Content Creation (Posts)
4. Content Discovery (Feed)
5. Basic Interaction (Likes, Comments)
6. User Following System
7. Scoring System for Social Engagement

## Primary User Journey
1. User signs up
2. User creates profile
3. User follows other users
4. User views feed of followed users
5. User creates posts
6. User interacts with content (like, comment)
7. User sees engagement scores

## Acceptance Criteria for MVP Validation
- All core features are implemented and functional
- Primary user journey flows seamlessly
- User can register, login, and logout
- Users can create and view posts
- Users can interact with posts (like, comment)
- Users can follow/unfollow other users
- Scoring system tracks engagement metrics

## Out of Scope (for MVP)
- Advanced analytics and reporting features
- Complex notification systems
- Multi-language support
- Push notifications
- Social media integration
- Advanced security features beyond authentication
