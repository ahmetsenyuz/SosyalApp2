# SosyalApp2 - Core User Journey Mapping

## Primary User Journey

### Phase 1: Onboarding
1. **User visits application**
   - Landing page with sign-up/login options
   - Guest access option

2. **User registration**
   - Provide basic information (username, email, password)
   - Account verification (if applicable)
   - Profile setup wizard

3. **User login**
   - Authenticate with credentials
   - Session management
   - Welcome screen

### Phase 2: Profile & Setup
4. **Profile customization**
   - Add profile picture
   - Personal information
   - Privacy settings

5. **Following others**
   - Search for users
   - Follow/unfollow functionality
   - View following list

### Phase 3: Content Engagement
6. **Viewing feed**
   - Timeline of followed users' posts
   - Sorting options (newest, popular, etc.)
   - Content filtering

7. **Creating content**
   - Post creation interface
   - Text, image, or video upload capability
   - Post visibility settings

8. **Interacting with content**
   - Like posts
   - Comment on posts
   - Share posts

### Phase 4: Social Metrics
9. **Engagement tracking**
   - View personal engagement score
   - See likes/comments received
   - Track follower growth

10. **Social impact measurement**
    - View post reach
    - See interaction statistics
    - Compare performance over time

## Key User Tasks
- Register new account
- Login/logout
- Create and manage profile
- Follow/unfollow users
- Create posts
- Interact with content (like, comment)
- View social feed
- Track personal engagement metrics

## Success Indicators
- Successful account creation and login
- Ability to create and view posts
- Smooth following/unfollowing experience
- Positive engagement with content
- Clear understanding of social metrics
