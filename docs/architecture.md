# SosyalApp2 Architecture

## Overview

This document describes the architectural approach for the SosyalApp2 social media application.

## Layers

1. **Presentation Layer** - Web UI/API
2. **Application Layer** - Business logic and orchestration
3. **Domain Layer** - Core business entities and rules
4. **Infrastructure Layer** - External dependencies and data access

## Technologies

- .NET 6.0
- Entity Framework Core
- ASP.NET Core Web API
- SQL Server
- JWT for authentication
